JSON Object

JSON (JavaScript Object Notation) là một định dạng dữ liệu phổ biến được sử dụng để truyền tải dữ liệu giữa các ứng dụng web và server. Object trong JSON được thể hiện bằng dấu ngoặc nhọn {}. Mỗi Object trong JSON chứa một hoặc nhiều cặp key-value, trong đó key và value được phân cách bởi dấu hai chấm : và các cặp key-value được phân cách bởi dấu phẩy ,. Key phải luôn nằm trong dấu ngoặc kép ", không được phép là biến số.

JSON Array

JSON Array là một tập hợp có thứ tự của các giá trị. Các giá trị có thể là bất kỳ kiểu dữ liệu JSON nào.
